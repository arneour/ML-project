# 📱 SMS Spam Detection - Machine Learning Pipeline

## Project Overview

This project builds a complete machine learning pipeline to classify SMS messages as **spam** (unwanted/promotional) or **ham** (legitimate). The pipeline includes exploratory data analysis, supervised learning, unsupervised clustering, and ensemble methods - all working together as a seamless workflow.

### Problem Statement
**Can we automatically detect spam messages using machine learning?** Spam messages cost time, enable fraud, and annoy users. This project aims to build an accurate classifier that can filter spam in real-time.

## 📊 Dataset

| Property | Value |
|----------|-------|
| **Name** | SMS Spam Collection |
| **Source** | UCI Machine Learning Repository |
| **URL** | https://archive.ics.uci.edu/ml/datasets/SMS+Spam+Collection |
| **License** | CC BY 4.0 |
| **Size** | 5,574 SMS messages |
| **Features** | label (spam/ham), message (text content) |
| **Class Balance** | 87% Ham, 13% Spam |

### Sample Data
| Label | Message |
|-------|---------|
| ham | "Ok then, I'll call you later" |
| spam | "WINNER!! Free iPhone! Call now!" |
