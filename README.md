# ML Project - Midterm

## About
This repository contains a machine learning project completed as part of a midterm assessment.  
It includes code, data analysis, and model building using Jupyter Notebook.

## Contents
- `Midterm_Project/` - main project folder
  - Jupyter Notebook with the full pipeline: from data loading to model evaluation
  - Data preprocessing, visualizations, and experiments with different algorithms

## Technologies Used
- **Language:** Python
- **Environment:** Jupyter Notebook
- **Libraries:**   - pandas (data manipulation)
  - numpy (numerical operations)
  - matplotlib (visualization)
  - seaborn (statistical visualizations)
  - scikit-learn (machine learning models, preprocessing, metrics)
